Project Overview

This summer I was part of a team that worked on the project, of developing frontend of registraion for user data app like addhar service where data is encrypted using private key with homomorphic encryption (microsoft SEAL) and is kept in a third party server.

Also had to develop another front end for verifying the information that is kept using user id that is send to the user at the time of registration. Data to be verified is also homomorphically encrypted and checked on the data already kept the answer is the of a vector that is decrpyted using the secret key.
